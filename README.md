# 🌍 GANS Cities — Data Cleaning & Urban Insights Engine

## 🧠 Executive Summary
This project transforms raw, inconsistent city-level data into a structured analytical dataset, ready for decision-making and future geospatial modeling.

Instead of focusing on "toy analysis", the project simulates a real-world data pipeline:
➡ messy input  
➡ cleaning & validation  
➡ structured output  
➡ insight generation  

---

## 🎯 Business Context
Urban and infrastructure decisions rely on clean, reliable data.

However, most real datasets suffer from:
- inconsistent formats
- missing values
- invalid entries
- lack of standardization

This project addresses exactly that problem.

---

## ⚙️ What This Project Does

### 1. Data Cleaning Pipeline
- Detects and removes invalid values
- Converts inconsistent formats into usable data types
- Handles missing data systematically
- Ensures dataset integrity

### 2. Data Validation
- Identifies anomalies and outliers
- Verifies consistency across fields
- Flags potential data quality risks

### 3. Exploratory Analysis (EDA)
- Distribution analysis
- Key metrics extraction
- Pattern identification

### 4. Dataset Structuring
- Prepares data for:
  - BI tools (Power BI / Looker)
  - Machine Learning models
  - Geospatial analysis (GIS)
